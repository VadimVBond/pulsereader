# Задача: разработка проекта PulseReader

Проект должен быть реализован с использованием Astro.

Перед началом разработки необходимо внимательно изучить файл PROJECT_CONTEXT.md и дизайн в папке DESIGN_REFERENCE.

Не начинать реализацию, пока архитектура не будет полностью понятна.

---

# ЭТАП 0 — Анализ

Перед разработкой необходимо:

1. Проанализировать файл PROJECT_CONTEXT.md
2. Проверить структуру проекта
3. Изучить дизайн из папки DESIGN_REFERENCE
4. Убедиться, что задачи реализуемы
5. Сообщить о возможных архитектурных улучшениях

После анализа подтвердить готовность к разработке.

---

# ЭТАП 1 — Инициализация проекта

Необходимо:

1. Инициализировать Astro проект
2. Настроить TypeScript
3. Подключить TailwindCSS
4. Создать базовую структуру папок
5. Настроить глобальные стили

На этом этапе страницы реализовывать не нужно.

---

# ЭТАП 2 — Базовая архитектура

Создать:

layouts
- BaseLayout.astro

components
- Header
- Footer
- Container

Все компоненты должны быть переиспользуемыми.

---

# ЭТАП 3 — Компоненты интерфейса

Реализовать:

ArticleCard

Компонент должен отображать:
- заголовок статьи
- краткое описание
- ссылку на статью

---

# ЭТАП 4 — Страницы

Создать страницы:

Главная  
src/pages/index.astro

Страница статьи  
src/pages/article/[slug].astro

---

# ЭТАП 5 — Контент

Использовать Astro Content Collections.

Структура:
src/content/articles

Статьи должны храниться в формате Markdown.

---

# ЭТАП 6 — SEO

Добавить:
- meta tags
- sitemap
- rss

---

# Общие требования

- использовать TypeScript
- не добавлять лишний JavaScript
- следовать архитектуре Astro
- писать чистый и модульный код
