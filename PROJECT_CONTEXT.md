# Контекст проекта PulseReader

PulseReader — это веб-проект для отображения и чтения статей.

Проект реализуется на Astro и ориентирован на:

- высокую производительность
- минимальное количество клиентского JavaScript
- статическую генерацию страниц

## Технологии

Основной стек:

- Astro 4
- TypeScript
- TailwindCSS
- React (используется только для интерактивных элементов)

## Архитектурные принципы

1. Максимально использовать серверный рендеринг Astro.
2. Минимизировать клиентский JavaScript.
3. Использовать компонентную архитектуру.
4. Хранить статьи через Astro Content Collections.
5. Использовать модульную структуру папок.

## Структура компонентов

components/

- Header
- Footer
- ArticleCard
- Container

layouts/

- BaseLayout

pages/

- Главная страница
- Страница статьи

## Дизайн

Оригинальный дизайн находится в папке:

DESIGN_REFERENCE

Верстка должна максимально соответствовать этому дизайну.

## Общие требования

- код должен быть чистым и читаемым
- использовать TypeScript
- следовать best practices Astro
- не добавлять лишние зависимости
