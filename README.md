# PulseReader

Стартовая структура проекта для разработки на Astro.

## Стек

- Astro
- TypeScript
- TailwindCSS
- React (Astro Islands)

## Папки

src/components — UI компоненты  
src/layouts — макеты страниц  
src/pages — страницы  
src/content — статьи  
src/styles — глобальные стили  

## Работа с Codex

2. Затем реализовывать проект по этапам.
